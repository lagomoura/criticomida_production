**Phase 0 decisions (2026-04-22):**
- Tokens/typography: Especiería + Cormorant + DM Sans is v2 (confirmed — docs reconciled to match code).
- Backend social endpoints do NOT exist yet. Strategy: mock-first via `NEXT_PUBLIC_SOCIAL_MOCK=true` flag. Helpers live in `app/lib/api/_mocks/index.ts` (`isSocialMockEnabled`, `mockDelay`). README at `app/lib/api/_mocks/README.md` lists missing endpoints.
- Route collision resolved: `/reviews/[category]` moved to `/categorias/[slug]` (param renamed too). 301 redirect added in `next.config.ts` for `/reviews/:slug` → `/categorias/:slug`. This frees `/reviews/[id]` for detail-of-a-social-post page.
